Craft My Plate

Project Overview

Overview

Craft My Plate is a test project built for an interview for a remote job in the food tech industry.

Features

  • Phone Auth: Easy and quick login with phone number.
  • Sleek and Modern UI: Designed for a modern user experience.

Setup Instructions

Prerequisites

  • Flutter installed on your machine. Follow the instructions here to install Flutter.

Steps

  1. Clone the Repository
    2. git clone https://github.com/Immadominion/craft-my-plate.git
    cd craft-my-plate
  2. Get Flutter Packages
    3. flutter pub get
  3. Run the App
    4. flutter run
